# Photo Organizer

This project is a simple application that copies photos from a source directory to a destination directory, organizing them by month and year based on their metadata.

## Features

- Copy photos from a specified source folder to a destination folder.
- Organize photos into subfolders by year and month of their creation date.

## Requirements

- Python 3.x
- Required libraries listed in `requirements.txt`

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/photo-organizer.git
   ```
2. Navigate to the project directory:
   ```
   cd photo-organizer
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Open `src/main.py` and set the `source_directory` and `destination_directory` variables to your desired paths.
2. Run the application:
   ```
   python src/main.py
   ```
